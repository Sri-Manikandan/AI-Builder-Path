# Approved Software

The following software is approved for use on Presidio devices without
additional sign-off.

## Productivity
- Microsoft 365 (Word, Excel, PowerPoint, Outlook, Teams)
- Google Chrome, Mozilla Firefox
- Slack
- Zoom

## Development
- Visual Studio Code
- JetBrains IDEs (license via IT)
- Docker Desktop
- Git, GitHub Desktop

## Security
- Cisco AnyConnect (VPN)
- CrowdStrike Falcon (endpoint protection — pre-installed, do not remove)
- 1Password (corporate password manager)

## Requesting new software

Software not on this list must be approved through the **IT Software Request**
form in the service portal. Requests are reviewed for security and licensing,
typically within 3 business days. Never install unapproved software from
unknown sources.
